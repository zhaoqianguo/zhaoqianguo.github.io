ES6 学习笔记（6） - Symbol

发表于2022-06-07|ES6
|阅读量:

Symbol

1. 概述

一种新的原始数据类型 Symbol ，表示独一无二的值。Symbol 值通过 Symbol()函数生成，也就是现在对象的属性名有两种类型，一种是字符串，一种是 Symbol 类型。

Symbol()函数可以接受一个字符串作为参数，表示对 Symbol 实例的描述。便于区分

💡 Symbol 值可以显示的转为字符串 和布尔值，但是不能转数值

2. Symbol.prototype.description

ES2019提供了一个 Symbol 值的实例属性description，直接返回 Symbol 值的描述。

3. 属性名的遍历

Symbol 值作为属性名，遍历对象的时候，该属性不会出现在for...in、for...of循中，也不会被Object.keys()、Object.getOwnPropertyNames()、JSON.stringify()返回。

可以使用 Object.getOwnPropertySymbols()

Reflect.ownKeys() 也可以

4. Symbol.for()，Symbol.keyFor()

  • Symbol.for()与Symbol()这两种写法，都会生成新的 Symbol。它们的区别是，前者会被登记在全局环境中供搜索，找到就会返回该值。后者不会，每次调用都会产生新值
  • Symbol.keyFor()方法返回一个已登记的 Symbol 类型值的key

Symbol.for()为 Symbol 值登记的名字，是全局环境的，不管有没有在全局环境运行。
